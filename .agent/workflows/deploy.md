# Deployment Workflow

Follow these steps to deploy your changes:

// turbo-all
1. Add all changes to the staging area:
   ```bash
   git add .
   ```

2. Commit the changes with a descriptive message:
   ```bash
   git commit -m "[change details]"
   ```

3. Push the changes to the remote repository:
   ```bash
   git push
   ```
